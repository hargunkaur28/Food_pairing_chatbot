# PairFect - Food Pairing Assistant 🍷🍽️

PairFect is a beautifully designed, responsive web application that acts as your personal artisanal sous-chef. Tell it your main ingredient, and it will curate the perfect recipes, pairings, and atmosphere for your meal.

## ✨ Features
- **Interactive Chat Interface**: A conversational UI simulating a personal sous-chef.
- **Smart Ingredient Matching**: Leverages the [Spoonacular API](https://spoonacular.com/food-api) to find matching ingredients and their culinary details.
- **Curated Recipes**: Recommends top-rated recipes based on your chosen ingredient.
- **Modern UI**: Features a vibrant, glassmorphism-inspired design with smooth animations and a responsive layout.

## 🚀 Deployment (Vercel)

This project consists of static HTML, CSS, and vanilla JavaScript, making it perfect for free hosting on [Vercel](https://vercel.com/).

Since the application files are located inside the `food_pairing` folder instead of the root directory, follow these steps:
1. Log into Vercel and click **Add New > Project**.
2. Import this GitHub repository (`hargunkaur28/Food_pairing_chatbot`).
3. Under **Configure Project**, edit the **Root Directory** setting and set it to `food_pairing`.
4. Click **Deploy**. Your app will be live in seconds!

## 🛠️ Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/hargunkaur28/Food_pairing_chatbot.git
   ```
2. Open the `food_pairing/index.html` file directly in your web browser, or use an extension like **Live Server** in VS Code.
3. Start exploring recipes!

## 🍽️ API Usage
This project uses the [Spoonacular Recipe API](https://spoonacular.com/food-api) to dynamically fetch and recommend meals.
